lotiR Documentation

Global temperature anomalies

Description

Anomalies, for the years 1880 to 2010, from the 1951 - 1980 average. These are the GISS (Goddard Institute for Space Studies) Land-Ocean Temperature Index (LOTI) data

Usage

loti

Format

A data frame with 131 observations on the following 19 variables.

Jan

a numeric vector

Feb

a numeric vector

Mar

a numeric vector

Apr

a numeric vector

May

a numeric vector

Jun

a numeric vector

Jul

a numeric vector

Aug

a numeric vector

Sep

a numeric vector

Oct

a numeric vector

Nov

a numeric vector

Dec

a numeric vector

J.D

Jan-Dec averages

D.N

Dec-Nov averages

DJF

Dec-Jan-Feb averages

MAM

Mar-Apr-May

JJA

Jun-Jul-Aug

SON

Sept-Oct-Nov

Year

a numeric vector

Source

http://data.giss.nasa.gov/gistemp/tabledata/GLB.Ts+dSST.txt

Examples

data(loti)
