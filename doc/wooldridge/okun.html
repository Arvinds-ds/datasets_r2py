okunR Documentation

okun

Description

Data loads lazily. Type data(okun) into the console.

Usage

data(okun)

Format

A data.frame with 47 rows and 4 variables:

  • year. 1959 through 2005

  • pcrgdp. percentage change in real GDP

  • unem. civilian unemployment rate

  • cunem. unem - unem[_n-1]
