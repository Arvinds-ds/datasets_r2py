wageprc

Description

Data loads lazily. Type data(wageprc) into the console.

Usage

data(wageprc)

Format

A data.frame with 286 rows and 20 variables:

  • price. consumer price index

  • wage. nominal hourly wage

  • t. time trend = 1, 2 , 3, ...

  • lprice. log(price)

  • lwage. log(wage)

  • gprice. lprice - lprice[_n-1]

  • gwage. lwage - lwage[_n-1]
